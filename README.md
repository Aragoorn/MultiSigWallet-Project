# Base MultiSig Wallet Infrastructure

This repository contains a robust Multi-Signature Wallet smart contract built with Solidity and TypeScript. It is designed to run efficiently on Ethereum Layer-2 networks, specifically **Base**.

## 🛠 Tech Stack
- **Smart Contract:** Solidity `^0.8.20`
- **Development Environment:** Hardhat
- **Language:** TypeScript
- **Testing:** Chai & Ethers.js v6

## 🚀 Features
- Propose, confirm, and execute multi-signature transactions safely.
- Ability to revoke confirmations before execution.
- Optimized for low gas fees on Layer-2 solutions.
